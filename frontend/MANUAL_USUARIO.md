# Manual do Usuário - Sistema Altave

## Sobre este Manual

Este documento descreve todas as funcionalidades do sistema Altave para usuários finais. Cada seção apresenta instruções passo a passo para utilização das principais funcionalidades.

---

## 1. Acesso ao Sistema

### 1.1 Cadastro de Novo Usuário

Para criar uma conta no sistema:

1. Acesse a página inicial do sistema
2. Clique em "Criar uma conta"
3. Preencha todos os campos obrigatórios:
   - Nome completo
   - E-mail corporativo
   - CPF (formato: 000.000.000-00)
   - Telefone (formato: (00) 00000-0000)
   - Data de nascimento
   - Senha (mínimo 6 caracteres)
   - Confirmação de senha
4. Marque a caixa "Li e aceito os termos de uso"
5. Clique em "Criar Minha Conta"
6. Aguarde a mensagem de confirmação
7. Você será redirecionado para a tela de login

**Observação**: O CPF e telefone são formatados automaticamente enquanto você digita.

**PRINT 1**: Tela de cadastro completa

---

### 1.2 Login no Sistema

Para acessar o sistema:

1. Acesse a página de login
2. Digite seu e-mail cadastrado
3. Digite sua senha
4. Opcionalmente, marque "Lembrar-me" para manter a sessão ativa
5. Clique em "Entrar no Sistema"

Após o login bem-sucedido:
- Usuários comuns são direcionados para seu perfil pessoal
- Administradores são direcionados para o dashboard

## 2. Perfil do Colaborador

### 2.1 Visualizar Perfil

O perfil contém cinco seções principais:

**Informações Básicas**
- Nome completo
- E-mail
- Cargo
- Telefone
- Data de nascimento

**Sobre Mim**
- Apresentação pessoal do colaborador

**Certificações**
- Lista de certificações profissionais obtidas

**Competências**
- Hard Skills: conhecimentos técnicos específicos
- Soft Skills: habilidades comportamentais

**Experiência Profissional**
- Histórico de posições anteriores

**PRINT 3**: Tela de perfil completa (visão geral)

---

### 2.2 Editar Perfil

Para modificar suas informações:

1. Acesse seu perfil
2. Clique no botão "Editar Perfil" (canto superior direito)
3. Os campos se tornarão editáveis
4. Modifique as informações desejadas:
   - Nome
   - Cargo
   - Telefone
   - Data de nascimento
   - Apresentação pessoal
5. Clique em "Salvar" para confirmar
6. Ou clique em "Cancelar" para descartar as alterações

**PRINT 4**: Tela de perfil em modo de edição

---

### 2.3 Gerenciar Competências

**Adicionar Hard Skills:**

1. Na seção "Hard Skills", localize o campo de entrada
2. Digite o nome da competência técnica
3. Clique no botão "+" ou pressione Enter
4. A competência será adicionada imediatamente

**Adicionar Soft Skills:**

1. Na seção "Soft Skills", localize o campo de entrada
2. Digite o nome da habilidade comportamental
3. Clique no botão "+" ou pressione Enter
4. A habilidade será adicionada imediatamente

**Remover Competências:**

1. Passe o mouse sobre a competência desejada
2. Clique no "X" que aparece
3. A competência será removida imediatamente

## 3. Minha Equipe

### 3.1 Visão para Colaboradores

Colaboradores comuns visualizam:

**Meu Supervisor**
- Nome, cargo e e-mail do supervisor direto
- Clique no card para visualizar o perfil completo

**Meus Colegas de Equipe**
- Lista de todos os membros da mesma equipe
- Nome, cargo e e-mail de cada colega
- Clique em qualquer card para ver o perfil

### 3.2 Visão para Supervisores

**Meu Diretor**
- Informações do diretor responsável

**Minha Equipe**
- Todos os colaboradores sob sua supervisão direta
- Contador de membros da equipe

**Outros Supervisores**
- Colegas que também são supervisores

**PRINT 7**: Tela Minha Equipe (visão supervisor)

**Todos os Colaboradores**
- Lista completa da organização

Esta funcionalidade está disponível apenas para administradores.

### 4.1 Visão Executiva

O dashboard apresenta:

**Métricas Principais**
- Total de colaboradores cadastrados

**Gráfico de Competências**

**PRINT 9**: Dashboard com métricas e gráfico

### 4.2 Buscar Colaboradores

Para localizar colaboradores:

1. No dashboard, clique em "Buscar Talentos"
2. Use a barra de pesquisa para filtrar por:
   - Nome
   - E-mail
   - Cargo
   - Competências
3. Clique em qualquer colaborador para ver detalhes completos
4. Use o botão "Voltar ao Dashboard" para retornar

**PRINT 10**: Tela de busca de colaboradores

## 7. Solução de Problemas

### Problemas Comuns

**Não consigo fazer login**
- Verifique se o e-mail está correto
- Confirme se a senha está correta (maiúsculas e minúsculas)
- Tente usar a recuperação de senha

**Alterações não são salvas**
- Verifique sua conexão com a internet
- Confirme que clicou em "Salvar" após editar
- Tente fazer logout e login novamente

**Não encontro um colaborador**
- Verifique a ortografia do nome na busca
- Confirme se o colaborador está cadastrado no sistema
- Entre em contato com o administrador
